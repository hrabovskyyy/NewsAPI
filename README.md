# 📰 NewsFinder Bot

> ⚠️ Цей проєкт ще перебуває на стадії активної розробки. Функціонал може змінюватися.

Telegram-бот для швидкого пошуку новин за допомогою публічного API. Підтримує категорії, пошук за ключовими словами, вибір країни, мови та щоденні розсилки.

## 🚀 Основні можливості:
- 🔍 Пошук новин за ключовими словами
- 🌍 Фільтрація новин за країною та мовою
- 🗂️ Категорії: технології, спорт, бізнес, наука тощо
- 📥 Отримання топ-5 / топ-10 новин
- 📎 Оцінка новин (лайк / дизлайк)
- 🕐 Автоматичні розсилки 1 раз на день
- 🔗 Посилання на повні статті
- 🇺🇦 Повністю україномовний інтерфейс

## 🛠️ Технології
- Python 3.11+
- Telegram Bot API (python-telegram-bot)
- REST API (наприклад, NewsAPI, GNews)
- Redis (опціонально — для збереження станів)
- Docker (опціонально)

## 📦 Встановлення
1. Клонуй репозиторій:
```bash
git clone https://github.com/твій-нік/newsfinder-bot.git
cd newsfinder-bot
```

2. Створи `.env` файл і вкажи токени:
```
TELEGRAM_TOKEN=твій_токен_бота
NEWS_API_KEY=токен_з_новинного_API
```

3. Встанови залежності:
```bash
pip install -r requirements.txt
```

4. Запусти бота:
```bash
python bot.py
```

## 📸 Приклад використання
- Користувач запускає бота
- Обирає тему "⚽ Спорт"
- Бот надсилає 5 найактуальніших новин зі зручними кнопками
